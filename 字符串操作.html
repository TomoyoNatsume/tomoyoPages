创建字符串

创建一个字符串变量的两种方式：
  • var str=new string('this is a string');
  • var str='this is a string';
注意js里创建数组的方法为：
  • var arr=new Array();
    arr[0]='123';
    arr[1]='456';
    ...
  • var arr=new Array('123','456',...);
  • var arr=['123','456',...];
其中前两种类似于创建字符串的第一种。

常用的特殊字符

大部分的标点符号，可以用反斜杠实现转义字符，如：
  • string str='I\'m Chinese.'
又因为js中既支持双引号，也支持单引号，所以在字符串中插入引号时，可以交替使用单引号或双引号，如：
  • string str="I'm Chinese."
虽然有时候连着打单引号/双引号也没有问题，但并不建议这样做。
\'单引号
\"双引号
\n换行
\\反斜杠
\r回车
\t制表符(tab)
\b退格(backspace)
\f换页

charCodeAt

返回在指定的位置的字符的 Unicode 编码（十进制）。如： 
var str='hello world';
str.charCodeAt(4);                  //返回第5个字母，即o的 Unicode 编码
字符串可以为中文、中英混合，且不考虑单双字节的问题。
对应的String.fromCharCode(65)是将65作为 Unicode 源码转为字符。

字符串与数字互相转换

1.转换函数
  • parseInt('123.3')：将'123.3'转为整数123，碰到小数点或任意非数字字符停止扫描
  • parseFloat('10.5')：将'10.5'转为浮点数
  • var num=12.345;
    var str=num.toString();     //返回字符串'12.345'
  • var num=12.345;
    var str=num.toFixed(2);     //返回字符串'12.34'
  • var num=12.345;
    var str=num.toPrecision(2);     //返回字符串'12'
  • var num=12.345;
    var str=num.toExponential();     //返回字符串'1.2345e+1'
2.强制类型转换
  • Boolean(参数)：这里的参数可以是字符串、数字，也可以是任何变量（对象）。
    只有在参数为空字符串、未定义、0、null这几种情况下才返回false
  • Number(参数)：可以传入字符串，如'5.5'，会返回5.5。但如果传入'5.5.5'，会返回NaN；
    传入null和false会返回0，传入true会返回1；
    传入对象或undefined会返回NaN；
  • String(参数)：把传入的任何变量/常量转换为字符串。如传入null，会返回'null'；传入true会返回true。
    实际上alert()函数的功能就与这个类似；
    如传入对象，则返回[object ***]
3.弱类型转换
  • var str='1234';
    var x=str-0;     //利用js弱类型

concat()

字符串拼接，如：
var string1='hello';
var string2='world';
var str=string1.concat(string2);     //str为'helloworld'
还有一种简单的拼接方法为直接相加，如： var str=string1+string2;

indexOf()

字符串检索。寻找目标字符串中第一次出现某字符串的位置，如：
var string='hello world';
var order=string.indexOf('wor');     //返回6，因为和数组一样从0开始
如果没有找到，则返回-1。
此外，lastIndexOf()是从后向前检索，使用方法一样。注意返回的位置是从前数的，即上面例子换为lastIndexOf返回值相同。
此外，includes()检索是否存在某字符串，使用方法一样。返回布尔值。
