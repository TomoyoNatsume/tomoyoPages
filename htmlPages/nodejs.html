创建服务器

最简单的一个服务器程序： 
    const http=require('http');
    var server=http.createServer(function(req,res){
    console.log(req.url);
    res.write("abc");
    res.end('hello world');
    });
    server.listen(8080);

回调函数

下面是一个回调实例： 
    var fs=require('fs');
    var data=fs.readFile('b1.txt',function(err,data){
        if(err)return console.log(err);
        console.log(data);
    });
    console.log('End');
这里会先输出'End'，然后输出data。 异步编程中，代码按顺序进行时，不会等某个读取文件的进程或某计算过程，而继续执行。所以在这里谁快谁输出。
node的文件系统中的每个方法都有同步和异步两个版本，这里文件读取的同步版本为readFileSync()。

事件机制

首先通过 
    var events=require('events');
来获取事件模块。而这个事件模块里只有一个对象：EventEmitter。然后执行下列代码： 
    var event = new events.EventEmitter(); 
    event.on('some_event', function() { 
        console.log('some_event 事件触发'); 
    }); 
    setTimeout(function() { 
        event.emit('some_event'); 
    }, 1000);
这个代码中，emit('some_event')用来发放事件，而event.on('some_event',fun)用来监听事件。fun为事件监听器的回调函数。
这里的发放事件函数和监听事件的回调函数都可以设置参数。前者的参数为字符串。
