    定义对象

    var person={name:"Jhon" sex:"male" height:"170"};
    其中name、sex、height为变量（属性），其后为对应的值。中间的空格可以换为逗号或换行。

    访问对象的变量：
    person.name
    或person["name"]
    对象方法： （相当于c语言里的成员函数）
        changeName:function(newName){
        return:newName;
    }
    这一段在定义对象时的代码的内部。而在外面调用函数时，和c一样：
    person.changeName("Jhonny");

    其他三种定义对象的方法

    • 使用new关键字
      先定义一个函数：(在定义函数的时候，系统会创建一个和函数名同名的类，所以可以使用new来创建对象。)
          function person(name,sex,height){
        this.name=name;//第一个name为这个类的一个属性，而第二个name为传入的形参
        this.sex=sex;
        this.height=height;
    }
    var p1=new person('Jhon','male','170');
    • 使用object类创建对象
      先定义一个空的object对象：(由于js为动态语言，因此可以动态地为该对象增加属性和方法) 
          var person=new object();
    person.name='Jhon';
    person.sex='male';
    • 使用json语法创建对象 即上面第一种定义对象的方法

    定义函数

    当用以下方法定义匿名函数时：
        var fun=function(){
        alert('fuck');
    }
    可以使用fun()来调用函数（为什么？）。此外，还可以在定义时加形参，调用时加实参。
    对于下面的方法定义的匿名函数：
        (function(m,n){
        alert(m+n);
    })(10,20);
    其实是定义+调用，也就是本身的内容运行了一遍。
    此外，这种方法还可以将第一个括号换为一元操作符，如+-*~!
        +function(m,n){
        alert(m+n);
    }(10,20);
    还有种写法：
        (function(m,n){
        alert(m+n);
    }(10,20));
    另外，在jQuery中，以下写法相当于$(document).ready(function(){...})： 
        $(function () {
        ...
    })

    运算符优先级

    关联性

    关联性决定了拥有相同优先级的运算符的执行顺序。考虑下面这个表达式： 
        a op b op c
    其中op代表运算符，如果是左关联，则相当于在a op b外面加了括号，先执行此步。而如果是右关联，则先执行b op c。最典型的，赋值运算符为右关联。

    返回值

    逗号表达式的一般形式是：表达式1，表达式2，表达式3……表达式n。逗号表达式的求解过程是：先计算表达式1的值，再计算表达式2的值，... 一直计算到表达式n的值，最后返回的值是表达式n的值。
    ...&&...的返回值并不是布尔值，而是左边或右边表达式的值。如果左边不为false，则返回左边，反之返回右边的值。如果是NaN或undefined等 导致的false，则也不会返回false，而是NaN或undefined等。'或'运算符'||'同理。

    练习

    1.判断下表达式的返回值，以及bol的最终值。（bol无初值） 
        1!==3&&(bol=7,3<9&&(bol-=3))
    虽然括号的优先级最高，但是这里括号的作用只是防止算成(..&&bol)，实际希望它算成...&&(bol-=3)。所以计算时，并不是先算最里面的括号 (bol-=3)，而是从左往右（按&&和===原本的关联性），先计算1===3，然后计算bol=7（而不是按优先级，最后才考虑赋值运算符），以此类推。 这里可以推理，如果不是这样的话，那么bol=7最后运算，bol-=3就会因为bol没有初值而报错。
    同样，可以试验，将前面改为1===3时，整个式子只判断到此就结束，最后bol会无定义。
    因此最终返回值为4（而不是true），bol=4。
    此外，试验以下表达式： 
        var a=1;
    alert(1==3&&a++);
