选择器

Query里使用$('p')类似的语句来定义对象，其中的'p'称为选择器。（可以为双引号） 下面是几种常用的选择器，其他的选择器见jQuery选择器
*选取所有元素
#idName选取id为idName的所有元素
.className选取类名为className的所有元素
p选取所有p元素
p:first选取所有p元素中的第一个p元素
p:last选取所有p元素中的最后一个p元素
p:even选取所有p元素中第偶数个p元素（注意索引值从0开始，所以这里是第1、3、5...个）
p:odd选取所有p元素中的奇数个p元素
p:first-child选取那些为其父元素的第一个子元素的所有p元素。注意，区分于下面一条选择器，此条选择器只选择那些p元素为所有子元素中第一个，前面不能有任何同胞元素
p:first-of-type选取那些为其父元素的第一个p子元素的所有p元素。此条选择器选出的p前面可以有其他任意同胞元素，只要不为p即可
注意当一个选择器为：
  • 'p ul'表示选取所有属于p元素的子元素的ul元素。这里的子元素包括直接和非直接的
  • 'p>ul'表示选取所有p元素的直接子元素中的ul元素。
  • 'p,ul'表示选取所有p元素或ul元素（逗号始终表示‘或’）
此外有选择器检测器。

设置属性

使用att()或toggleClass('className')切换元素的类名。在jQuery中toggle的用法基本上是“有就减去，没有就加上”。这里也一样，如果元素已经是这个 类名了，就去掉，如果不是该类，就设为该类。如果元素除了这个类还有其他类，其他类不会受影响。

css

使用.css()函数设置样式，如： 
          $('p').css('background-color','red');
          或
          $('p').css({
            "background-color":"red",
            "color":"blue"
          });
注意第二种方法中两句之间要用逗号，而不是分号。而且每个属性及其值都要用引号（单双都行）

尺寸

遍历

  1. parent()返回被选元素的直接父元素
  2. parents()返回所有祖先元素
    可以添加参数如'li'指返回所有祖先元素中的li元素
  3. parentsUntil('ul')返回直到ul元素之间的所有祖先元素。当然，不包括自己和ul，即双开区间
  4. children()返回所有直接子元素
    与parents()类似，也可添加参数来进行筛选（下面有类似的函数时不再赘述）
    注意，table的直接子元素为tbody，要得到tr元素的话还需要再返回一次子元素。
  5. find('li')返回所有后代中的li元素。
    用'*'来返回所有元素
  6. siblings()返回所有同胞元素，即属于同一个直接父元素的其他元素
  7. next()返回下一个同胞元素
  8. nextall()返回后面的所有同胞元素
  9. nextUntil('ul')返回本元素和ul之间的所有同胞元素，也是双开区间
  10. prev()、prevAll()、prevUntil()与上面next对应

过滤

  1. first()返回JQuery对象(数组)中的第一个对象 如$('p div').first() 返回$('p')
  2. last()与上面对应
  3. eq(n)返回第n+1个元素
  4. 如$('p').eq(2)等同 注意区别于$('p')[2]; 后者返回的是选择器选出来的第三个元素的DOM对象，而前者返回的是这个元素的jQuery对象
  5. filter('.className')返回带有className类名的元素
  6. not('.className')返回不带有className类名的元素

Ajax

  1. load(URL,data,callback)返回URL里的内容到对象中。
  2. $.get(URL,callback)返回得到的数据。回调函数里有两个回调参数，第一个参数存放数据内容，第二个存放请求状态
  3. $.post(URL,data,callback)用于请求向服务器提交数据。
